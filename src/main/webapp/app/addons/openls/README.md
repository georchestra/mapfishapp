OpenLS ADDON
===============

author: @fvanderbiest

The addon config should look like this:

    {
        "id": "openls_0",
        "name": "OpenLS",
        "title": {
            "en": "OpenLS recentering",
            "es": "OpenLS recentering",
            "fr": "Recentrage OpenLS"
        },
        "description": {
            "en": "OpenLS recentering",
            "es": "OpenLS recentering",
            "fr": "Recentrage OpenLS"
        },
        "options": {
            "serviceURL": "http://my.server.tld/openls"
        }
    }

The serviceURL option is mandatory.
